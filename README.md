🏥 Healthcare Appointment & Follow-up Manager
Project Overview

Healthcare Appointment & Follow-up Manager is an AI-powered healthcare platform that streamlines appointment scheduling, symptom analysis, follow-up care, and patient-doctor communication. The system provides dedicated dashboards for patients, doctors, and administrators while integrating Large Language Models (LLMs) to generate intelligent pre-visit and post-visit summaries.
